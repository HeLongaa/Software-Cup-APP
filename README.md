## 安装和本地启动  
  
- 本项目运行于Python-3.11环境下  
  
  
1. 本地启动  
     
   >    git clone https://github.com/HeLongaa/Software-Cup-APP.git  
    >   
    >    cd Software-Cup-APP  
    >   
    >    pip install requirements.txt  
    >   
    >    set FLASK_APP=app.py  
    >   
    >    flask run
